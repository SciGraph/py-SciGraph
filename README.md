Python API and Command Line Client for [SciGraph](https://github.com/SciGraph/SciGraph/)

## Examples

### Autocomplete

    ./run-scigr.py  a Parkinson

### Graph Visualization

    ./run-scigr.py -t png  g OMIM:118300


### Annotation

    ./run-scigr.py ann "the big ears and the hippocampus neurons"
